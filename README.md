# youtube-playlist-downloader
This tool can download entire youtube playlists...

# Installation
+ You need to install python first.
+ Install `pytube` :
``` sh
pip install pytube
```
+ Install `tqdm` :
``` sh
pip install tqdm
```

# Usage
You need to run the script using python : 
``` sh
python main.py
```
